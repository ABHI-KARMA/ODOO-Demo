Library Management System

Supported Editions
Community
Enterprise

Library management system is an app that is designed to manage library. It helps librarians to maintain the data of new books and the books that are borrowed by members along with their due dates.

Key Features

Book Master

Librarian can add books manually in library
Librarian can add books by ISBN.

Member Portal

Member can issue a request for available books.
If Book is not available in the library, a member can reserve books.
Member can check book requests & books reservation status.

Member Operations Management Through Web.

Member can issue book request.
Member can reserve book.
Member can view book request & reservation records.

Well Integrated with Inventory Management.

Stock will automatically handle when book is issued & returned.

Reporing

Librarian can print multiple book issue history in a pdf format.

Role Management

Two roles first is Librarian (All Backend Management) and second is Member (User Portal Rights)

Librarian Operations Management Through Backend.

Librarian can create Member's Membership
Librarian can approve/disapprove book request & reservation request.

Membership Management

Librarian can create Paid Membership or Free Membership.
Librarian can print Membership card.

Emailer

Email Notification.

Set Library Management System

  • Settings >> Users & Companies >> Users >> Library >> Librarian,Member.
  • Settings >> Users & Companies >> Users >> Technical >> Manage Multiple Stock Locations.
  • If Library is Private library Admin can active Paid Membership in configuration and set Price
  • Set Due Date Exceed Amount and Due Date in Settings >> Configuration
Book Master

  • Add book in library by ISBN.
  • Add book manually by librarian.
Book Library

  • Member can Filter Available Books and not available books.
Book Request

  • Click on Issue Request button.
  • Member is submiting form.
  • Librarian get a mail.
  • Open My Account and check book request status.
Approve Book Request

  • Librarian get a book issue request
  • Librarian Approve Book issue Request.
  • Member get a mail with reject issue request date.
Issue Book Request

  • Set picking type id and confirm book issue.
  • Book is issued.
  • Book state is done.
Reject Book Request

  • Librarian reject book request with a reason.
  • Member gets a book request rejection mail.
  • Book state is rejected.
Return Book

  • Set picking type and return book.
  • Member get a book return mail.
Reissue Book

  • Member want to reissue book.
  • Book is reissued.
Book Reservation

  • When book is out of stock, Member wants to reserve books.
  • Member is submiting book reservation form.
  • Librarian get a mail with reservation number.
  • Librarian check book reservation request and update book stock.
  • When book is in stock librarian can issue book.
Membership

  • Create Membership
Paid Membership

  • Set membership price depends on time.
  • Membership Card
